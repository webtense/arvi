# Recursos de marca ARVI

Actualizado el 10 de abril de 2026.

## Ubicación principal
- Carpeta: `Facturas /ARVI LOGO, ETC`

## Archivos relevantes
- `original_logo_arvi.pdf`: Logo vectorial en alta calidad. Ideal para plantillas PDF y material impreso.
- `JAUMEARVI2.pdf`, `JAUMEARVI3.pdf`, `JAUMEARVIGRIS.pdf`, `JAUMEARVI.pdf`: Variantes con fondo blanco/gris, útiles para fondos oscuros o claros.
- `logo_arvi_alta.jpg`, `logo_arvi_baja.jpg`: Versiones rasterizadas de alta/baja resolución para uso rápido en la web.
- `ORIGINAL_LOGO.zip`: Paquete comprimido con todos los recursos anteriores.
- `SOBRE CON LOGO.docx`: Plantilla de sobre/cubierta con logotipos y datos corporativos.
- `Conforme seguro bó.pdf.jyhyf3i.partial`: Archivo descargado parcialmente (puede eliminarse si no se completa la descarga).

## Recomendaciones de uso
1. **Generación de PDFs (facturas, partes, presupuestos)**: incrustar `original_logo_arvi.pdf` o `JAUMEARVI3.pdf` para mantener nitidez en cualquier tamaño.
2. **Sitio web / emails**: usar `logo_arvi_alta.jpg` y exportar versión recortada a WebP para optimizar peso.
3. **Copias de seguridad**: conservar `ORIGINAL_LOGO.zip` en almacenamiento seguro junto con esta documentación.
4. **Limpieza**: revisar el archivo parcial `Conforme seguro bó.pdf.jyhyf3i.partial`; si la descarga ya no es necesaria, eliminarlo para evitar confusiones.
